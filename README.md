Projeto Tradutor Web

Este projeto consiste em uma aplicação web de tradução de texto, desenvolvida com foco em praticar lógica de programação, JavaScript e manipulação do DOM.
A aplicação permite ao usuário inserir um texto, selecionar o idioma de origem e o idioma de destino, retornando a tradução de forma simples e intuitiva.

O projeto foi desenvolvido com tecnologias web básicas, priorizando código limpo, organização e facilidade de entendimento, sendo ideal para fins de estudo, portfólio e evolução contínua.

Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

Objetivos do projeto

* Praticar lógica de programação
* Trabalhar com eventos e funções em JavaScript
* Manipular elementos da interface (DOM)
* Criar uma aplicação funcional e intuitiva
